
 # College Management System

## Step 1: Create Custom Objects

### 1. Student (`Student__c`)

Fields:

* **Name** → Standard Name field
* **Roll_Number__c** → Number
* **Email__c** → Email/Text
* **Marks__c** → Number

### 2. Faculty (`Faculty__c`)

Fields:

* **Faculty_Name__c** → Text
* **Faculty_ID__c** → Number (**Unique**)
* **Salary__c** → Currency
* **Department__c** → Picklist
* **Joining_Date__c** → Date

---

# Step 2: Validation Rules

Go to:

**Setup → Object Manager → Select Object → Validation Rules → New**

---

# Student Validation Rules

## 1. Validate Marks

### Formula

```plaintext id="xjlvwo"
OR(Marks__c < 0, Marks__c > 100)
```

### Error Message

```plaintext id="u6k4ya"
Marks must be between 0 and 100.
```

---

## 2. Validate Roll Number

### Formula

```plaintext id="f90jhl"
Roll_Number__c <= 0
```

### Error Message

```plaintext id="c68zxr"
Roll Number must be greater than 0.
```

---

## 3. Validate Email

### Formula

```plaintext id="7h4c3r"
NOT(CONTAINS(Email__c, "@"))
```

### Error Message

```plaintext id="7l8cqa"
Please enter a valid email address.
```

---

# Faculty Validation Rules

## 1. Validate Faculty Name

### Formula

```plaintext id="0x6xwl"
OR(
   ISBLANK(Faculty_Name__c),
   LEN(Faculty_Name__c) < 3
)
```

### Error Message

```plaintext id="nyqul6"
Faculty Name must contain at least 3 characters.
```

---

## 2. Validate Faculty ID

### Formula

```plaintext id="j7tltu"
Faculty_ID__c <= 0
```

### Error Message

```plaintext id="lln4hv"
Faculty ID must be greater than 0.
```

---

## 3. Validate Salary

### Formula

```plaintext id="sld7di"
OR(
   Salary__c <= 10000,
   Salary__c >= 500000
)
```

### Error Message

```plaintext id="xwz1y0"
Salary must be greater than 10,000 and less than 500,000.
```

---

## 4. Validate Department

### Formula

```plaintext id="yz2pcg"
ISPICKVAL(Department__c, "")
```

### Error Message

```plaintext id="4h9b9e"
Please select a Department.
```

---

## 5. Validate Joining Date

### Formula

```plaintext id="o2prg0"
Joining_Date__c > TODAY()
```

### Error Message

```plaintext id="qq5ew0"
Joining Date cannot be a future date.
```

# Step 3: Create Visualforce Pages

Go to:

**Setup → Visualforce Pages → New**

Enable:

* **Available for Lightning Experience**
* **Experience Builder Sites**
* **Mobile App**

---

## StudentRegistration Page

```xml id="jlm36y"
<apex:page standardController="Student__c" lightningStylesheets="true">
    <apex:form>
        <apex:pageMessages/>
        
        <apex:pageBlock title="Student Registration">
            <apex:pageBlockSection columns="2">
                <apex:inputField value="{!Student__c.Name}"/>
                <apex:inputField value="{!Student__c.Roll_Number__c}"/>

</apex:page>
```

# Step 4: Create Visualforce Tabs

Visualforce pages cannot directly appear in the navigation bar unless tabs are created.

Go to:

**Setup → Tabs**

Scroll down to the **Visualforce Tabs** section and click **New**.

---

## Create Student Registration Tab

1. Select **StudentRegistration** page from the dropdown.
2. Enter:

   * **Tab Label:** Register Student
   * **Tab Name:** Register_Student
3. Choose a suitable tab style/icon.
4. Click **Next**.
5. Keep profile visibility as default.
6. Click **Next → Save**.

---

## Create Faculty Registration Tab

Repeat the same steps:

1. Select **FacultyRegistration** page.
2. Enter:

   * **Tab Label:** Register Faculty
   * **Tab Name:** Register_Faculty
3. Choose another icon/style.
4. Click **Next**.
5. Keep default profile settings.
6. Click **Save**.

These tabs make the Visualforce pages accessible from the Lightning App navigation menu.
