🧾 Google App Engine Deployment (Flask + Gunicorn) – Exam Steps
🔹 Step 1: Create New Project
Go to Google Cloud Console
Click Select Project → New Project
Enter name (e.g., gcloudtrial)
Click Create
🔹 Step 2: Enable App Engine
Go to App Engine
Click Create Application
Select region (e.g., asia-south1)
App Engine is initialized
🔹 Step 3: IAM Permissions (VERY IMPORTANT)
Go to IAM & Admin → IAM

Find service account:

project-id@appspot.gserviceaccount.com
Add roles:
App Engine Admin
Storage Admin
Cloud Build Editor

👉 This step fixes deployment errors.

🔹 Step 4: Initialize gcloud CLI
gcloud init

Choose:

Account → your Gmail
Project → your created project
🔹 Step 5: Verify Setup
python --version
gcloud config list
gcloud app describe
🔹 Step 6: Create Project Folder
mkdir gcloudtrial
cd gcloudtrial
🔹 Step 7: Create Files
✅ main.py
from flask import Flask

app = Flask(__name__)

@app.route('/')
def home():
    return """
    <html>
        <body style="background-color:black; color:yellow; text-align:center; font-family:Arial;">
            <h1>Hello, Google App Engine!</h1>
            <p>Deployment successful 🎉</p>
        </body>
    </html>
    """
✅ requirements.txt
flask
gunicorn
✅ app.yaml
runtime: python311
entrypoint: gunicorn -b :$PORT main:app
🔹 Step 8: Deploy Application
gcloud app deploy

Press:

Y
🔹 Step 9: View Output
gcloud app browse

👉 Output:

Hello, Google App Engine!
Deployment successful 🎉
🔹 Step 10: Debug (if error occurs)
gcloud app logs tail -s default
