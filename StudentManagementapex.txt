Step 1: Create Custom Object

Go to:

Setup → Object Manager → Create → Custom Object

Fill the details:

Label: Student
Plural Label: Students
Object Name: Student
Record Name: Student Name

Click Save

Important:

Salesforce automatically creates:

Name

field by default.

So do NOT create Name field separately.

Step 2: Create Custom Fields

Create these 3 fields:

Field Label	Data Type
Rollno	Number
Class	Text
Phoneno	Phone

Click Save for each field.

Step 3: Create Apex Class

Go to:

Setup → Apex Classes → New

Paste this code:

public class StudentController {

    public Student__c s {get; set;}

    public StudentController() {
        s = new Student__c();
    }

    public List<Student__c> getStudents() {
        return [
            SELECT Id, Name, Rollno__c, Class__c, Phoneno__c
            FROM Student__c
        ];
    }

    public void saveStudent() {
        insert s;
        s = new Student__c();
    }

    public void deleteStudent() {
        delete [
            SELECT Id
            FROM Student__c
            WHERE Id = :ApexPages.currentPage()
                .getParameters().get('id')
        ];
    }
}


Click Save

Step 4: Create Visualforce Page

Go to:

Setup → Visualforce Pages → New

Page Name:
StudentRecord
<apex:page controller="StudentController">
    <apex:form>

        <apex:pageBlock title="Student Management">

            <apex:pageBlockSection columns="1">

                <apex:inputField value="{!s.Name}"/>
                <apex:inputField value="{!s.Rollno__c}"/>
                <apex:inputField value="{!s.Class__c}"/>
                <apex:inputField value="{!s.Phoneno__c}"/>

                <apex:commandButton 
                    value="Save"
                    action="{!saveStudent}"
                    rerender="t"/>

            </apex:pageBlockSection>

        </apex:pageBlock>

        <apex:pageBlock title="Student Records" id="t">

            <apex:pageBlockTable value="{!students}" var="x">

                <apex:column value="{!x.Name}" headerValue="Student Name"/>
                <apex:column value="{!x.Rollno__c}" headerValue="Roll No"/>
                <apex:column value="{!x.Class__c}" headerValue="Class"/>
                <apex:column value="{!x.Phoneno__c}" headerValue="Phone No"/>

                <apex:column headerValue="Action">

                    <apex:commandLink 
                        value="Delete"
                        action="{!deleteStudent}"
                        rerender="t">

                        <apex:param 
                            name="id"
                            value="{!x.Id}"/>

                    </apex:commandLink>

                </apex:column>

            </apex:pageBlockTable>

        </apex:pageBlock>

    </apex:form>
</apex:page>
Click Save

Step 5: Run the Page

Open this URL:

/apex/StudentRecord

Now the system will:

Save Student Records
Display Student Records
Delete Student Records
